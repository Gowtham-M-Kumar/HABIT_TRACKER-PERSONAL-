Product vision
What this app is
A personal habit dashboard that makes consistency visual and satisfying. One beautiful screen. No friction. Check off habits, watch your data grow in real time.
Who it's for
Students, professionals, anyone building daily routines — people who want a calm, beautiful tool they'll actually open every day, not a cluttered app they abandon.
The single design promise
Every number and chart on screen updates the moment you tap a checkbox. No save buttons. No loading. The dashboard is always alive — like watching your life score in real time.

Animations and live interactions
Every interaction that animates
Checkbox tap
Cell pops with a spring scale(1.2) → scale(1) bounce. Color fills with habit's unique color. All rings/charts update with a 300ms ease transition simultaneously.
Progress rings
SVG stroke-dashoffset animation — ring "draws" itself smoothly every time a number changes. Daily ring, habits ring, and all 5 week rings are live.
Bar chart bars
Each bar animates height from 0 on initial load (stagger 30ms per bar). On data change, bars smoothly grow or shrink.
Area chart wave
The mountain silhouette in the header redraws with a smooth path morph when data changes. Fills a soft lavender blue.
Month switch
Entire dashboard fades out (150ms), data swaps, fades back in (150ms). Month title crossfades with the new name.
Progress bars
Habit progress bars in the right panel animate width with CSS transition whenever count changes.
Leaderboard re-sort
When ranking changes, rows animate to new positions with a smooth translateY transition.
Number count-up
Any time a stat number increases, it counts up from old value to new in 400ms (requestAnimationFrame loop).

Data model — what gets stored
localStorage JSON schema
habits[ ]
Array of habit objects: id, name, color, goal (target days/month), icon emoji, createdAt, active: true/false.
logs{ year: { month: { habitId: { day: boolean } } } }
Nested object keyed by year → month → habitId → day number → true/false. Very lightweight. A full year of 20 habits ≈ 15KB.
profile{ }
name, affirmation text, photoDataURL (base64 thumbnail), theme, accentColor.
settings{ }
darkMode, startDayOfWeek, showWeekends, notifications (future), appVersion for migration.

Build phases — development roadmap
Phase 1 — Foundation Day 1
Vite + React + Tailwind setup. PWA config. Zustand store. localStorage data layer. Habit CRUD (add/edit/delete). Basic grid rendering.
Phase 2 — Core grid and interactions Day 2
31-day habit grid. Tap-to-toggle checkboxes with animation. Day header row (M T W T F S S). Color-coded per habit. Live stat calculation engine.
Phase 3 — Charts and rings Day 3
SVG animated progress rings (daily + habits + 5 weekly). Recharts weekly bar charts. Recharts area chart header. All live-update on checkbox tap.
Phase 4 — Right panels + leaderboard Day 4
Top 10 leaderboard with animated re-sort. Daily progress table with bars, count, streak. Streak calculation algorithm. Motivational quote logic.
Phase 5 — Polish and settings Day 5
Settings drawer. Profile photo upload. Affirmation editor. Month/year navigation with full data persistence. Dark mode toggle. Framer Motion page transitions.
Phase 6 — Deploy Day 6
Mobile responsive layout. PWA manifest + service worker. Install prompt on Android. Deploy to Vercel. Test on Mac browser + Android. Done.
